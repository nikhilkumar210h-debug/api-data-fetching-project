# API Data Fetching and Data Collection with Python

## 📌 Project Overview

This project demonstrates how to collect real-world data from APIs using Python. The notebook fetches data from public APIs, processes JSON responses, converts them into structured datasets using Pandas, and exports the results to CSV files for further analysis.

It is a beginner-friendly project designed to help learners understand API integration, data collection, and basic data processing workflows used in data analysis.

---

## 🎯 Objectives

- Learn how APIs work
- Send HTTP requests using Python
- Fetch and process JSON data
- Convert raw API responses into tabular datasets
- Export data into CSV format
- Build a simple data collection pipeline

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- Requests

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/nikhilkumar210h-debug/api-data-fetching-project.git
```

### Navigate to Project Directory

```bash
cd api-data-fetching-project
```

### Install Required Libraries

```bash
pip install pandas requests and bs4
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open:

```text
api_data_fetch.ipynb
```

and run all cells.
